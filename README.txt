This program is a user interface that allows to manage a list of polygonal shapes.

The list can contain a maximum of 100 objects that inherit the class Polygon.

The user has 5 options:

1 - Create a Rectangle
2 - Create a Rhombus
3 - Create a Right Triangle
4 - Draw all the items in the list
5 - Exit

option 1: create and add a Rectangle to the list
option 2: create and add a Rhombus to the list
option 3: create and add a Right Triangle to the list
option 4: print area and perimeter of all the items in the list
option 5: delete all the items in the list and exit the program

When creating an object (options 1-2-3) the user is asked to insert the dimension of it.

If the list is full (limit of 100 objects reached) options 1-2-3 will stop adding objects noticing the user that the list is full.
